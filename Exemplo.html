HTML: O que é isso?

DOCUMENTO HTML!

HTML significa HyperText Markup Language; como o próprio nome diz, é uma linguagem de marcação usada para definir e descrever a estrutura de uma página da web.

TAG

HTML usa tags para marcar cada seção de um documento.

ELEMENTOS HTML

Um Elemento HTML é criado marcando algum conteúdo com uma tag de abertura, colocada antes do conteúdo, e uma tag de fechamento colocada após o conteúdo, ambas as tags usando o mesmo nome. Embora a maioria dos elementos use um par de tags, existem alguns elementos que não precisam de uma tag de fechamento (eles também não possuem um conteúdo), esses elementos são chamados de elementos vazios. Podemos colocar um elemento dentro de outros elementos (não vazios), aninhando vários níveis de elementos um dentro do outro, ao fazer isso devemos ter o cuidado de fechar todos os elementos que criamos dentro do outro antes de fechar o elemento central.

Todos os documentos HTML consistem em elementos HTML aninhados.

ATRIBUTOS

Além do nome do elemento, as tags de abertura podem consistir em alguns outros valores que fornecem informações adicionais sobre um elemento. Esses valores são chamados de atributos.
